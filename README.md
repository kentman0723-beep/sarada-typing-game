# 皿打 (Sara-Da) - タイピングゲーム

小悪魔から食べ物を守れ！Flashゲーム「皿打」にインスパイアされたタイピングゲームです。

## 🎮 遊び方

1. 難易度を選択（EASY / NORMAL / HARD）
2. 小悪魔が皿に向かって飛んでくる
3. 吹き出しに表示されたローマ字をタイピング
4. 小悪魔を撃退して食べ物を守ろう！

## 🎯 操作方法

- **a-z キー**: タイピング入力
- **Backspace**: 入力取り消し
- **Escape**: 入力リセット

## ✨ 特徴

- 3段階の難易度
- 200語以上の単語データ
- スコア＆コンボシステム
- 効果音（Web Audio API）
- レスポンシブデザイン

## 🖥️ 技術スタック

- HTML5 Canvas
- Vanilla CSS3
- Vanilla JavaScript
- Web Audio API
